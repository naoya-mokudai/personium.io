Personium Documentation Contributor Guide

Personium Project welcomes your contribution to all documentation.
Please follow the following instructions to add or modify Personium documentation. Personium Project members will review and apply your changes accordingly.

GitHub account is needed

  1. Fork your own copy of Personium repository to your account.
  2. Clone the repository to your working environment.
  3. Please edit in markdown format.
  4. About HTML file.
    • If you want to reflect your changes to HTML immediately, please follow this procedure to convert your document from markdown to HTML.
    • If you leave it unchanged, it will be automatically converted after we approved your pull request.
  5. Commit and push your changes to your forked copy.
  6. Create a pull request to merge your changes to the original reposity.
